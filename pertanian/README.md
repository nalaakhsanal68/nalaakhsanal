

# 🌾 Platform Pertanian Indonesia

Website sederhana dan mudah digunakan untuk mendukung petani Indonesia dengan informasi lengkap tentang pertanian modern.

## ✨ Fitur Utama

### 1. 💰 **Harga Pasar Real-time**
- Monitor harga padi, jagung, sayuran, dan tomat
- Lihat tren harga (naik/turun/stabil)
- Update data dari pasar lokal

### 2. 📚 **Edukasi Pertanian**
- **Teknik Menanam**: Panduan lengkap persiapan lahan, pemilihan bibit, jarak tanam
- **Irigasi & Manajemen Air**: Jenis irigasi, jadwal penyiraman, konservasi air
- **Hama & Penyakit**: Identifikasi dan pengendalian hama organik
- **Pupuk Organik**: Cara membuat pupuk sendiri dari sampah rumah tangga

### 3. 🌤️ **Perkiraan Cuaca**
- Prakiraan cuaca 4 hari ke depan
- Data suhu dan kelembaban
- Membantu perencanaan pertanian

### 4. 📰 **Berita Pertanian**
- Kebijakan pemerintah terbaru
- Teknologi pertanian terkini
- Pergerakan harga pasar
- Workshop dan pelatihan gratis

### 5. 🚀 **Teknologi Pertanian**

- 🔬 Alat Pengukur Kualitas
- ❄️ Sistem Penyimpanan Dingin

## 🚀 Cara Menggunakan

### 1. **Buka Website**
- Download atau clone repository ini
- Buka file `index.html` di browser favorit Anda
- Website siap digunakan!

### 2. **Navigasi**
- Gunakan menu di atas untuk navigasi antar section
- Klik "Mulai Sekarang" untuk scroll ke Harga Pasar
- Semua section dapat diakses dengan smooth scrolling

### 3. **Pelajari Edukasi**
- Klik tombol "Pelajari" di setiap kartu edukasi
- Modal akan muncul dengan panduan lengkap
- Tutup modal dengan tombol X atau tombol Escape

### 4. **Cek Informasi**
- Lihat harga pasar dan trend
- Monitor perkiraan cuaca
- Baca berita pertanian terbaru
- Explore teknologi pertanian modern
